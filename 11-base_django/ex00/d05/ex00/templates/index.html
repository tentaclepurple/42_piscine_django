Ex00: Markdown Cheatsheet

1. Encabezados (Headers)

  • # Encabezado 1 →

    Encabezado 1

  • ## Encabezado 2 →

    Encabezado 2

  • ### Encabezado 3 →

    Encabezado 3

  • #### Encabezado 4 → Encabezado 4
  • ##### Encabezado 5 → Encabezado 5
  • ###### Encabezado 6 → Encabezado 6

2. Énfasis (Emphasis)

  • *Texto en cursiva* o _Texto en cursiva_ → Texto en cursiva
  • **Texto en negrita** o __Texto en negrita__ → Texto en negrita
  • ***Texto en negrita y cursiva*** → Texto en negrita y cursiva

3. Listas (Lists)

Listas no ordenadas (Unordered Lists)

- Elemento 1
* Elemento 2
- Elemento 3
  • Elemento 1
  • Elemento 2
  • Elemento 3

Listas ordenadas (Ordered Lists)

1. Primer elemento
2. Segundo elemento
3. Tercer elemento
  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

4. Citas (Blockquotes)

> Esta es una cita.
Esta es una cita.

5. Código (Code)

Código en línea (Inline Code)

Para insertar código en línea, usa `código`, por ejemplo: `print("Hello, World!")`.

Bloques de código (Code Blocks)

Usa tres acentos graves (```) para crear un bloque de código:

    ```
    def hola_mundo():
        print("¡Hola, Mundo!")
    ```

8. Tablas (Tables)

Las tablas se crean usando | y - para definir las columnas y filas:

    | Encabezado 1 | Encabezado 2 |
    |--------------|--------------|
    | Celda 1      | Celda 2      |
    | Celda 3      | Celda 4      |
Encabezado 1 Encabezado 2
Celda 1 Celda 2
Celda 3 Celda 4

10. Escapando caracteres especiales (Escaping Special Characters)

Para mostrar caracteres especiales (como * o #) sin formateo, usa una barra invertida (\) delante del carácter.

  • \*Texto\* muestra *Texto* en lugar de Texto.
